## 🎨 个人自我介绍网站项目计划

### 技术栈

* **前端**: Vue 3 + Nuxt 3 + Tailwind CSS

* **后端**: Node.js + Express + JWT认证

* **数据库**: MySQL

* **版本控制**: Git + GitHub

* **部署**: 阿里云服务器 (47.251.101.229)

***

### 📁 项目结构

```
self-intro/
├── frontend/          # Nuxt3 前端项目
│   ├── pages/         # 页面（首页、管理后台）
│   ├── components/    # 组件（内容块、编辑器等）
│   ├── composables/   # 组合式函数
│   └── assets/        # 静态资源
│
├── backend/           # Express 后端项目
│   ├── routes/        # API路由
│   ├── middleware/    # 中间件（认证等）
│   ├── models/        # 数据模型
│   └── uploads/       # 上传文件存储
│
└── deploy/            # 部署配置
```

***

### 🚀 实现步骤

#### 阶段一：项目初始化与GitHub配置

1. 初始化Git仓库，创建 `.gitignore`
2. 在GitHub创建远程仓库
3. 初始化 Nuxt 3 前端项目，配置 Tailwind CSS
4. 初始化 Express 后端项目，连接 MySQL
5. 设计数据库表结构（内容块、媒体文件、点赞记录）

#### 阶段二：前台展示页面

1. 实现炫酷可爱的首页布局（动画效果、渐变背景）
2. 开发多形态内容组件：
   * 📝 文字/富文本块
   * 🖼️ 图片展示块
   * 🎬 视频播放块
   * 📄 PDF嵌入块
3. 实现点赞/喜欢功能（带可爱动画）

#### 阶段三：后台管理系统

1. 实现登录页面 + JWT认证
2. 开发内容管理界面：
   * 可视化拖拽排序
   * 富文本编辑器
   * 媒体文件上传
3. 布局配置功能

#### 阶段四：部署上线

1. 推送代码到GitHub
2. 配置 PM2 进程管理
3. Nginx 反向代理配置
4. 部署到阿里云服务器
